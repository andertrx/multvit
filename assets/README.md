## Diretrizes de Fotografia

### Iluminação
- Luz natural ou luz suave difusa
- Evitar sombras duras
- Fundo bem iluminado

### Composição
- Produtos centralizados
- Espaço negativo generoso
- Foco nítido no produto

### Cores
- Fundos neutros: branco (#FFFFFF), bege claro (#F5F3EF), cinza claro (#F3F4F6)
- Harmonia com a paleta da marca
- Evitar cores muito saturadas

### Estilo
- Minimalista e limpo
- Profissional mas acessível
- Inspiração: fotografia de produto premium

## Como Adicionar as Imagens

1. Salve as imagens neste diretório (`/assets/`)
2. Mantenha os nomes exatamente como listado acima
3. Otimize as imagens antes de fazer upload (recomendado: TinyPNG ou Squoosh)
4. As imagens serão carregadas automaticamente no site

## Otimização de Imagens

Para melhor performance, otimize suas imagens:

1. **Formato**: Use WebP quando possível (fallback para JPG)
2. **Tamanho**: Não exceda 200KB por imagem
3. **Dimensões**: Redimensione para o tamanho recomendado
4. **Ferramentas**: TinyPNG, Squoosh, ImageOptim
